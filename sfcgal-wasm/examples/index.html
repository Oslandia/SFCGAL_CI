About SFCGAL

SFCGAL is a C++ wrapper library around CGAL with the aim of supporting ISO 19107:2019 and OGC Simple Features for 3D operations. These examples demonstrate real SFCGAL algorithms compiled to WebAssembly, running directly in your browser.
