SFCGAL provides a basic 3D viewer based on OpenSceneGraph. The following video shows how we can use it to make it interact with a PostGIS database.

This is an overview of what is possible to achieve with SFCGAL, an SFCGAL-enabled version of PostGIS and a version of QGIS that supports 3D visualization.
